6. 您的权利 / Your Rights (GDPR Art. 15-21)

📋 访问权 Right of Access (Art. 15 + Art. 20 Portability)

获取我们持有的所有关于您的数据副本（JSON 格式）。

行使方式: GET /api/privacy/export（需 X-Device-Id 请求头）

🗑 删除权 Right to Erasure / "Right to Be Forgotten" (Art. 17)

请求删除所有个人数据（支付记录按法规保留）。请求提交后 30 天内执行。

行使方式: POST /api/privacy/erase

🚫 限制处理权 Right to Restrict Processing (Art. 18)

暂停对您数据的处理（分析/个性化），同时保留数据。

行使方式: POST /api/privacy/restrict | 撤销: DELETE /api/privacy/restrict

✏ 更正权 Right to Rectification (Art. 16)

通过应用内设置随时更新您的旅行偏好和保存地点。

🔔 撤回同意 Withdraw Consent (Art. 7)

随时撤回同意，不影响撤回前处理的合法性。

行使方式: POST /api/privacy/consent 中设置 "granted": false

您有权向所在欧盟成员国的数据保护监管机构（DPA）投诉。中国境内用户可联系国家互联网信息办公室（CAC）。

8. 安全措施 / Security Measures

  • 传输加密: 所有 API 通信使用 TLS 1.2+
  • 数据最小化: LLM 调用前自动 PII 脱敏
  • 访问控制: RBAC 分区（b2c / b2b / admin）
  • 审计日志: 所有操作含 SHA-256 完整性哈希
  • 安全响应头: CSP、X-Frame-Options、X-Content-Type-Options、Referrer-Policy、Permissions-Policy
  • 输入防护: 8 种提示注入模式自动拦截
  • 速率限制: 每 IP 每分钟 10-30 次请求上限
  • IP 哈希: 同意记录存储 SHA-256 IP 哈希，不存储原始 IP
