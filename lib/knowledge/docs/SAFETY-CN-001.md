# 在华旅行安全与紧急情况指南

## 重要紧急电话

### Q: 在中国遇到紧急情况应该拨打哪个电话？
A: 中国紧急服务电话（所有手机均可拨打，无需SIM卡信号也可能通话）：

| 服务 | 号码 | 说明 |
| :--- | :--- | :--- |
| 报警（警察）| **110** | 盗窃/抢劫/意外事故 |
| 火警 | **119** | 火灾/火灾救援 |
| 急救（救护车）| **120** | 医疗紧急情况 |
| 交通事故 | **122** | 交通事故报警 |
| 旅游投诉热线 | **12301** | 旅游纠纷/导游投诉 |

**使用中文报警**：拨打110后，可说"我是外国游客，我需要帮助"，警方通常会找英语翻译协助。

## 医疗急救

### Q: 在中国生病或受伤了怎么办？
A:
**就医流程：**
1. **轻伤/小病**：直接前往附近药店（药房，绿色+字标志）购买OTC药品，药剂师会推荐
2. **普通就医**：前往"三甲医院"（最高等级医院），设有国际部（VIP部门）
3. **紧急情况**：拨打120急救，或立即前往最近医院急诊室（急诊/急救）

**外国游客友好医院（有英语服务）：**
- **北京**：北京协和医院国际医疗部、北京和睦家医院（Beijing United Family Hospital）
- **上海**：上海仁济医院、上海国际医疗中心（SIMC）、和睦家医院
- **广州**：广州中山一院国际部

**医疗费用参考（2026年）：**
- 普通门诊（三甲医院）：¥50–300（挂号费+诊疗费）
- 急诊（含检查）：¥300–1,000
- 国际部/和睦家：¥800–3,000/次诊（无需中国医保）
- **强烈建议**：出发前购买含医疗费用的旅游保险，中国不承认外国医保

## 常见安全风险

### Q: 在中国旅行有哪些常见安全风险需要注意？
A: 中国整体安全水平高（街头暴力犯罪率低），但需注意以下：

**旅游区常见骗局：**
- **茶馆骗局（Tea House Scam）**：在北京/上海景区，陌生人邀请"去喝茶/看书法展览"，最终索要高额费用；礼貌拒绝即可
- **出租车绕路**：拒绝非正规出租车，使用滴滴打车（如同Uber），显示路线和价格
- **换汇黑市**：使用正规银行/ATM换汇，拒绝街头换汇
- **假冒景点讲解员**：进入景区前，官方导游有证件；私人"讲解员"收费不透明

**个人财物安全：**
- 地铁/公交/人流密集处注意手机和钱包（扒手多见于游客区）
- 使用酒店保险箱存放护照/大额现金
- 手机不要放背包外侧口袋；背包在人群中建议背到前面

## 数字安全与VPN

### Q: 在中国使用手机和网络有什么限制？
A:
**中国防火长城（Great Firewall）屏蔽的服务：**
- Google全套（Gmail/Maps/搜索）
- WhatsApp、Facebook、Instagram、Twitter/X
- YouTube
- ChatGPT/Gemini（AI服务）

**解决方案：**
- **出发前**在国内下载并配置好VPN（ExpressVPN/NordVPN等）
- 进入中国后部分VPN会被干扰，7–8月党代会/敏感时期管控更严
- **中国可用的替代品**：百度地图（导航）、微信（即时通讯）、微博（社交）、高德地图

**手机SIM卡：**
- 在机场或中国电信/移动/联通门店购买本地SIM卡，需护照实名登记
- 当地SIM流量套餐约¥50–100/月（30GB流量起）
- 也可开通香港SIM卡（部分翻墙限制较少）

## 护照丢失处理

### Q: 在中国护照丢失了怎么办？
A:
1. **立即报警**：持中文"我的护照丢失了"（Wǒ de hùzhào diūshīle）到最近派出所报案，取得"报案回执"
2. **联系本国大使馆/领事馆**：通过大使馆申请紧急旅行证件（一般1–3个工作日）
3. **中国主要城市大使馆联系方式**：
   - 大使馆列表查询：China.cn → 外交部领事服务网（consular.mfa.gov.cn）
   - 美国大使馆（北京）：+86-10-8531-4000
   - 英国大使馆（北京）：+86-10-5192-4000
   - 澳大利亚大使馆（北京）：+86-10-5140-4111
   - 马来西亚大使馆（北京）：+86-10-6532-2531
4. **临时证明**：凭报案回执+大使馆紧急证明，可在中国境内临时住宿和购票
