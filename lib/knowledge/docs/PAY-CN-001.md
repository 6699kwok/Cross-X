# 外国游客在华移动支付完整指南

## 微信支付 (WeChat Pay) 境外卡绑定

### Q: 外国人如何将境外银行卡绑定到微信支付？
A: 截至 2026年2月，外国游客可通过以下步骤将 Visa 或 Mastercard 绑定至微信支付：
1. 打开微信 → 点击右下角"我" → "服务" → "钱包"。
2. 点击"银行卡" → "添加银行卡"。
3. 输入持卡人姓名（必须与护照姓名完全一致）、卡号、有效期、CVV 及绑定手机号。
4. 完成手机验证码验证后即绑卡成功。
- **支持卡种：** Visa、Mastercard（部分 Maestro 卡不支持）。
- **单笔消费限额：** 截至 2026年2月，境外卡单笔限额为 **6,250 元人民币**。
- **每月消费限额：** 截至 2026年2月，境外卡每月累计限额为 **50,000 元人民币**。

### Q: 微信支付境外卡绑定失败的常见原因和解决方案是什么？
A:
| 失败原因 | 解决方案 |
| :--- | :--- |
| 银行拒绝境外交易 | 提前致电发卡行开通"境外网络支付"功能 |
| 姓名不匹配 | 确保输入姓名与护照首页完全一致（全大写拉丁字母） |
| 手机号无法接收验证码 | 改用 +60 (马来西亚) 或 +1 (美国) 等境外手机号，确保能收国际短信 |
| 卡片类型不支持 | 联系银行确认该卡是否支持在线 CNP (Card Not Present) 交易 |

## 支付宝 (Alipay) 境外用户使用指南

### Q: 外国游客应该使用哪个版本的支付宝，与国内版本有何区别？
A: 外国游客应下载并使用**支付宝国际版（Alipay International）**，而非中国大陆版本。
- **账号注册：** 使用境外手机号（如 +60 马来西亚号码）直接注册，无需中国身份证。
- **绑卡方式：** 支持直接绑定 Visa/Mastercard/American Express 境外信用卡。
- **使用场景：** 可在中国大陆的绝大多数商家（餐厅、超市、地铁、打车）扫码消费。
- **限额（截至 2026年2月）：** 单笔限额约 **5,000 元人民币**，每日限额约 **20,000 元人民币**。

### Q: 支付宝的"TourCard"（旅行卡）是什么，如何使用？
A: 支付宝 TourCard 是专为短期来华外国游客推出的虚拟预付卡服务。
1. 打开支付宝国际版 → 搜索"TourCard" 或 "境外游客支付"小程序。
2. 使用境外信用卡充值（最低充值 ¥100，最高 ¥5,000）。
3. 充值后获得一张虚拟的人民币预付卡，可在中国所有支持支付宝的商户使用。
- **优势：** 无需实名认证，开通快，适合短期旅游（有效期 90 天）。
- **到期退款：** 90 天内未消费完的余额，可申请退回至原绑定境外卡。

## 境外银行卡在华 ATM 取现

### Q: 外国游客在中国境内使用境外银行卡在 ATM 取现，手续费是多少？
A: 请参考以下各银行 ATM 政策表格（截至 2026年2月）：

| ATM 运营银行 | 是否支持境外卡 | 参考手续费 | 单次取现上限 |
| :--- | :--- | :--- | :--- |
| 中国银行 (Bank of China) | ✅ 支持 | 约 ¥20-30 + 发卡行收费 | ¥5,000/次 |
| 中国工商银行 (ICBC) | ✅ 支持 | 约 ¥20-30 + 发卡行收费 | ¥5,000/次 |
| 中国建设银行 (CCB) | ✅ 支持 | 约 ¥20-30 + 发卡行收费 | ¥5,000/次 |
| 招商银行 (CMB) | ✅ 支持 | 约 ¥20 + 发卡行收费 | ¥3,000/次 |
| 便利店 ATM（7-11 等） | ⚠️ 部分支持 | 更高，需确认 | ¥2,000/次 |

- **重要提示：** 实际手续费由你的**发卡行**和 ATM 运营行**双重收取**，出境前请咨询你的发卡银行。
